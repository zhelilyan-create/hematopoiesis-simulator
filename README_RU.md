# Симулятор дифференциации клеток — v0.1

Минимальный модульный симулятор дифференциации клеток гемопоэза на Python,
построенный на основе непрерывной марковской цепи (CTMC / алгоритм Гиллеспи).

> **ВНИМАНИЕ: все значения скоростей в `configs/hematopoiesis_v01.yaml`
> являются заглушками, выбранными исключительно для структурного
> тестирования. Они НЕ получены из экспериментальных данных.
> Не интерпретируйте результаты моделирования как количественные
> биологические предсказания.**

---

## Что делает этот симулятор

- Моделирует гемопоэз как CTMC по 8 дискретным типам клеток
- Три типа событий для каждой клетки: **дифференциация**, **симметричное
  деление самообновления**, **апоптоз**
- Прямой метод Гиллеспи (точное стохастическое моделирование)
- Детерминированная воспроизводимость через фиксацию случайного зерна (seed)
- Запись временных рядов состава популяции
- Базовая визуализация (динамика популяции во времени, итоговый состав)

### Иерархия типов клеток (v0.1)

```
ГСК (HSC)
└── МПП (MPP)
    ├── ОМП (CMP)
    │   ├── Миелоидная     (терминальная)
    │   └── Эритроидная    (терминальная)
    └── ОЛП (CLP)
        ├── B-клетка       (терминальная)
        └── T-клетка       (терминальная)
```

Терминальные типы не имеют исходящих событий дифференциации или деления в v0.1.

---

## Чего этот симулятор НЕ делает

| Функциональность | Статус |
|---|---|
| Внутреннее состояние клетки (уровни белков, фаза клеточного цикла) | Не реализовано |
| Асимметричное деление клеток | Не реализовано |
| Возраст центриолей / отслеживание материнских и дочерних центриолей | Не реализовано |
| Пространственное моделирование | Не реализовано |
| Генная регуляторная сеть | Не реализовано |
| Обратная связь на уровне популяции (напр., ёмкость среды) | Не реализовано |
| Биологически калиброванные скорости | Не реализовано — все скорости являются заглушками |

## Требования

```
Python >= 3.10
numpy >= 1.26
pandas >= 2.0
pyyaml >= 6.0
matplotlib >= 3.5   (опционально, для --plot-out)
pytest >= 7.4       (опционально, для тестов)
```

Установка:

```bash
pip install numpy pandas pyyaml
pip install matplotlib          # опционально, для визуализации
pip install pytest              # опционально, для тестов
```

## Как запустить

### Базовое моделирование (100 ч, случайное зерно)

```bash
python scripts/run_sim.py
```

### Воспроизводимый запуск с фиксированным зерном

```bash
python scripts/run_sim.py --t-max 200 --seed 42
```

### Сохранение итоговой диаграммы

```bash
python scripts/run_sim.py --t-max 200 --seed 42 --plot-out results.png
```

## Запуск тестов

```bash
pytest tests/
```

Тесты покрывают:

- Разбор конфигурации — валидная, отрицательные скорости, цели `None`, неизвестные типы клеток
- Консистентность популяции — изменение размера после апоптоза, дифференциации, деления
- Поведение движка — детерминированная воспроизводимость, остановка по `t_max`,
  семантика начального снимка, исключение событий с нулевой скоростью,
  ограничения событий терминальных типов клеток

## Структура проекта

```
cell_diff_sim/
  cell.py                 Датакласс Cell + псевдоним CellType
  population.py           Контейнер популяции
  models/
    base.py               Интерфейс AbstractModel + протокол InheritanceRules
    hematopoiesis.py      Модель гемопоэза v0.1 (8 состояний)
    inheritance.py        DefaultInheritanceRules (тривиальные, v0.1)
  engine/
    events.py             DifferentiationEvent, DivisionEvent, ApoptosisEvent
    cell_factory.py       create_daughter() — единственный путь создания дочерних клеток
    division_handler.py   DivisionHandler — оркестрация деления
    ctmc.py               CTMCEngine — прямой метод Гиллеспи
  observers/
    recorder.py           Recorder + Snapshot — запись временных рядов
  analysis/
    plots.py              plot_population_over_time, plot_final_composition
configs/
  hematopoiesis_v01.yaml  Параметры скоростей (ЗАГЛУШКИ — НЕ КАЛИБРОВАНЫ)
scripts/
  run_sim.py              Точка входа
tests/
  test_config.py          Тесты разбора конфигурации
  test_population.py      Тесты консистентности популяции
  test_engine.py          Тесты поведения движка
```

## Планы на v0.2

Следующая версия добавит (в порядке приоритета):

1. **Типизированный датакласс `InternalState`** — заменит временную заглушку
   `internal_state: dict` в `Cell`. Будет содержать уровни белков, фазу
   клеточного цикла и т.д.
2. **Состояние центриолей** — датакласс `CentrioleState`; логика наследования
   в подклассе `CentrioleInheritanceRules`.
3. **Асимметричное деление** — `DivisionEvent.daughter_cell_types` уже содержит
   кортеж из 2 элементов; `InheritanceRules.inherit(daughter_index=0/1)` уже
   имеет соответствующий слот. Нужна лишь новая реализация на стороне модели —
   движок не изменяется.
4. **Зависимость вероятностей судьбы от состояния** —
   `model.get_events(cell, population)` уже получает полную популяцию; скорости
   могут быть сделаны функциями от `cell.internal_state` без каких-либо
   изменений движка.
